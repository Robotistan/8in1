|               ENGLISH (EN)              |
+-----------------------------------------+
Welcome to the Robotistan INC flashing tool! This tool allows you to easily flash MicroPython onto your ESP32 or REX Board.

1- Connect your ESP32: Plug your ESP32 or REX Board into your computer via USB.
2- Run the uploadFirmware.bat file: Double-click the .bat file to run it.
3- Select your COM port: When prompted, type the COM port (for example: COM3).
4- The tool will automatically erase the flash and write MicroPython to your device.
	- If you see the message "Failed to get PID of a device on COM*, using standard reset sequence." 
	and dots keep appearing continuously ("...."), press and hold the "BOOT" button on your device and 
	try again to upload the firmware.
5- Once done, the process is complete!

That's it! You’re ready to start using MicroPython on your ESP32!

Important Notes:
Windows Warning: If a Windows warning appears when you run the .bat file, click More Info and then click Run Anyway to proceed with the flashing process.
COM Port not detected: If you don't see your device in the COM port list, you may need to install the CH340 driver. Run the CH341SER.EXE file and click Install to install the necessary drivers.
